# ResetGuest

[![Language](https://img.shields.io/badge/Language-Kotlin-blue)](https://kotlinlang.org)

Aplikasi Android untuk mereset akun guest Mobile Legends via root shell dengan satu ketukan. Menghapus shared preferences ML.

## Requirements

- Android 8.0+ (API 26)
- Root access (Magisk / KSU)
